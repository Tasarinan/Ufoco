# Install

First, fork the repo and clone to your local

## Development

Install dependencies

```sh
npm install
```

Run `webpack-dev-server`

```sh
npm run dev
```

Open another terminal session and start the app

```sh
npm run start
```

## Production

Install dependencies

```sh
npm install
```

Compile scripts

```sh
npm run compile
```

Pack the app

```sh
npm run pack
```

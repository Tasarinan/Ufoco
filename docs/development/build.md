# Build

We only support the stable version of Node.js. Current stable version is `v10.*.*`.

## Development

Install dependencies

```sh
npm install
```

Run `webpack-dev-server`

```sh
npm run dev
```

Open another terminal session and start the app

```sh
npm run start-dev
```

## Production

Install dependencies

```sh
npm install
```

Compile scripts

```sh
npm run compile
```

Pack the app

```sh
npm run pack
```
